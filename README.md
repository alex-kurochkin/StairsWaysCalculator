# StairsWaysCalculator

Задача: найти количество способов (путей) подняться по одной или нескольким лестницам, если допустимо пропускать несколько ступеней. По-сути комбинаторная задача о Числах Фибоначчи.
Тестовый код для решения задачи. Впрочем, код написан не ради решения самой задачи, сама задача решается куда меньшим объёмом кода.

Конструктор StairsWaysCalculator принимает параметр $possibilities - максимальное количество возможных пропусков ступеней.
Метод StairsWaysCalculator::calculateWaysTotal() принимает целые числа - количество ступеней в каждой лестнице.
